# HostsMe
------
- ## 简介 
> **HostsMe** 是一个用JAVA Swing写的小工具，可以通过修改本地Hosts文件的方法，以达到**科学上网**的目的，无需任何额外设置，无需**VPN**，访问速度取决于本地网络环境，不用再忍受龟速代理
